Installation/Instructions for use:
The R/Sweave script "rnaseq.rnw" can be run in any R/Sweave compatible editor (ie. emacs-ess or RStudio). All necessary source data files have been included in the appropriate directory structure.
Knit/Sweaving the script will reproduce majority of figures for Yung et al.

Expected output:
Report with embedded figures as in "rnaseq.pdf"

< 30 minutes
